2단계 문자열 계산기
===

## 요구사항
* 사용자가 입력한 문자열 값에 따라 사칙연산을 수행할 수 있는 계산기 구현
  * 사칙연산에 해당하는 덧셈/뺄셈/곱셈/나눗셈 을 연산하는 계산기 구현
* 연산자의 우선순위가 아닌 입력순서에 따라 계산
  * ex) "2 + 3 * 4 / 2"는 연산자 우선순위가 아닌 입력 순서에 따라 실행결과인 10을 출력
